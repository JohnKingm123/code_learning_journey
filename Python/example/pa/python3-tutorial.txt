Python 3 教程
Python 的 3.0 版本，常被称为 Python 3000，或简称 Py3k。相对于 Python 的早期版本，这是一个较大的升级。为了不带入过多的累赘，Python 3.0 在设计的时候没有考虑向下相容。
Python 介绍及安装教程我们在Python 2.X 版本的教程中已有介绍，这里就不再赘述。
你也可以点选 Python2.x与3??.x版本区别 来检视两者的不同。
本教程主要针对 Python 3.x 版本的学习，如果你使用的是 Python 2.x 版本请移步至 Python 2.X 版本的教程。
官方宣布，2020 年 1 月 1 日， 停止 Python 2 的更新。
检视 Python 版本
我们可以在命令视窗(Windows 使用 win+R 调出 cmd 执行框)使用以下命令检视我们使用的 Python 版本：
python -V 或 python --version
以上命令执行结果如下：
Python 3.3.2
你也可以进入Python的互动式程式设计模式，检视版本：
Python 3.3.2 (v3.3.2:d047928ae3f6, May 16 2013, 00:03:43) [MSC v.1600 32 bit (Intel)] on win32 Type "copyright", "credits" or "license()" for more information. >>>
第一个Python3.x程式
对于大多数程式语言，第一个入门程式设计程式码便是"Hello World！"，以下程式码为使用Python输出"Hello World！"：
例项(Python 3.0+)
#!/usr/bin/python3 print("Hello, World!")

执行例项 ?
你可以将以上程式码储存在 hello.py 档案中并使用 python 命令执行该指令码档案。
$ python3 hello.py
以上命令输出结果为：
Hello, World!
